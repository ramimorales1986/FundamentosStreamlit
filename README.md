# 🚀 Fundamentos de Streamlit

Este repositorio contiene una aplicación desarrollada con **Streamlit** como parte del aprendizaje de los fundamentos para la creación de aplicaciones web interactivas en Python.

## 📌 Objetivo

El propósito de este proyecto es demostrar el uso de los componentes básicos de Streamlit, incluyendo:

- Creación de interfaces web.
- Uso de widgets interactivos.
- Entrada de datos del usuario.
- Barra lateral (Sidebar).
- Visualización dinámica de información.
- Personalización de la aplicación.

## 📂 Estructura del Proyecto

```
FundamentosStreamlit/
│
├── streamlit.py      # Aplicación principal
├── LogoUHE.png       # Logo utilizado en la interfaz
└── README.md         # Documentación del proyecto
```

## 🛠️ Tecnologías utilizadas

- Python 3.x
- Streamlit

## 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/ramimorales1986/FundamentosStreamlit.git
```

Ingresa al directorio:

```bash
cd FundamentosStreamlit
```

Instala las dependencias:

```bash
pip install streamlit
```

## ▶️ Ejecutar la aplicación

Desde la terminal ejecuta:

```bash
streamlit run streamlit.py
```

La aplicación se abrirá automáticamente en tu navegador.
